# Installation and Setup

## Database Setup
1. Run `sql/schema_creation.sql` to create object types
2. Run `sql/type_bodies.sql` to implement methods
3. Run `sql/table_creation.sql` to create tables
4. Run `sql/data_insertion.sql` to populate sample data

## Query Testing
Run `sql/queries.sql` to test all demonstration queries
