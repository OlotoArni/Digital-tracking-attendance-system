# Digital Attendance Management System - Project Overview

## Project Description
An automated attendance tracking system for York University using object-oriented database design with Oracle.

## Key Features
- Object-oriented database design with inheritance
- MAP, ORDER, and OVERRIDE methods implementation
- Device-based attendance tracking
- Real-time enrollment and capacity management

## Technology Stack
- Database: Oracle with Object-Relational features
- Object Types: User inheritance hierarchy
- Methods: MAP, ORDER, OVERRIDE implementations
