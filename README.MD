# Digital Attendance Management System (DAMS)

## Project Overview
Automated attendance collection system for York University using object-oriented database design.


## Course
AP/ITEC 4220A - Database Management Systems Fall 2025  
Professor: Marina G Erechtchoukova

## Project Structure
- `docs/` - Project documentation and PDF submissions
- `sql/` - Database schema and queries
- `presentation/` - PowerPoint slides
- `diagrams/` - Class diagrams and other visuals

## Milestone 2 Features
- Object-oriented database design
- Inheritance hierarchy for User types
- MAP, ORDER, and OVERRIDE methods implementation
- Attendance tracking with device integration
